<h1 align="center">HashZap</h1>

## 🚀 Introduction

  The HashZap Python project develops a live chat similar to WhatsApp, using Flask and Flask-SocketIO. Users can exchange messages in real-time, without access to the conversation history, providing a dynamic and interactive experience.

## 🌐 Functionalities:
- Real-time message exchange: Users can send and receive messages instantly, enabling quick and effective communication.
- No conversation history: There is no access to message history, ensuring privacy and promoting an authentic live chat experience.

## 🛠️ Technologies Useds:
- Flask: A micro web framework for Python used to build the web application.
- Flask-SocketIO: An extension for Flask that enables WebSocket support, facilitating real-time communication between clients and the server.
- Simple Websocket: A library for WebSocket communication, providing a simple interface for establishing WebSocket connections.
- HTML/CSS: Used for creating the user interface and styling the web pages.
  
## 🐱‍🚀 How to use
- set socket to “localhost:5000”
- set filename to app
- Inside the terminal you will write python –m flask run
- As soon as you run the code in the terminal, you'll see some messages, including a link. This link is where your chat page is located, so just click on it to open the chat in your browser.
